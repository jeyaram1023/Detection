How to Use

Camera On/Off: Use the 📷 button to start or stop your webcam. Please grant camera permissions when prompted.

Flashlight: The 💡 button will turn on your phone's flashlight if supported. Otherwise, it will flash the screen white.

Switch Mode: Use the 🔁 button in the menu to toggle between detecting all objects and only specific animals.

Snapshot: Capture the current view with detections using the 🖼️ button.
